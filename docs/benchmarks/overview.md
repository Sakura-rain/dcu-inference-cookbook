## 测试方法

### LLM / VLM

- **吞吐量 (Tokens/s)**: 使用随机 prompt，测量每秒生成 token 数
- **首 Token 延迟 (TTFT)**: 从请求到第一个 token 的时间
- **并发测试**: 多并发请求下的性能表现

### 图像生成

- **生成速度**: 单张图片生成时间（秒/张）
- **吞吐量**: 每分钟生成图片数

### 语音模型

- **实时率 (RTF)**: 处理时间 / 音频时长，RTF < 1 表示实时
